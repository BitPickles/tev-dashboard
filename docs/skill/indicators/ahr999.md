# AHR999 指标设计

## 指标概述

| 属性 | 值 |
|------|-----|
| 名称 | AHR999 (九神指标) |
| 路径 | `/ahr999/index.html` |
| 数据 | `indicators/data/ahr999.json` |
| 作者 | 九神 |
| 用途 | 比特币长期估值，判断定投时机 |

## 计算公式

```
AHR999 = (当前价格 / 200日定投成本) × (当前价格 / 币龄估值)
```

### 200日定投成本
过去 200 天 BTC 价格的**几何平均值**（非算术平均）

### 币龄估值
```
估值 = 10^(5.84 × log10(币龄天数) - 17.01)
```
币龄天数 = 当前日期 - 2009-01-03 (创世区块)

## 状态区间

| AHR999 范围 | 状态 | 颜色 | 建议 |
|-------------|------|------|------|
| < 0.45 | 抄底区 | 🟢 绿色 | 历史低估，可加仓 |
| 0.45 - 1.2 | 定投区 | 🟡 黄色 | 正常定投 |
| > 1.2 | 观望区 | 🔴 红色 | 价格偏高，谨慎 |
