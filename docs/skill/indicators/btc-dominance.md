# BTC Dominance 指标设计

## 指标概述

| 属性 | 值 |
|------|-----|
| 名称 | BTC.D (Bitcoin Dominance) |
| 路径 | `/btc-dominance/index.html` |
| 数据 | `indicators/data/btc-dominance.json` |
| 用途 | 比特币市值占比，判断资金流向和市场周期 |

## 计算公式

```
BTC.D = BTC 市值 / 加密货币总市值 × 100%
```

## 指标含义

| BTC.D 趋势 | 含义 |
|------------|------|
| 上升 | 资金从山寨币流向 BTC，避险情绪 |
| 下降 | 资金从 BTC 流向山寨币，风险偏好上升 |
| 极高 (>65%) | BTC 主导，山寨币弱势 |
| 极低 (<40%) | 山寨币季节，泡沫风险 |

## 周期规律

典型牛市周期：
1. **早期**：BTC.D 上升，BTC 领涨
2. **中期**：BTC.D 稳定，ETH 和主流山寨开始表现
3. **末期**：BTC.D 快速下降，山寨币狂欢
4. **熊市**：BTC.D 回升，山寨币暴跌更多

## 状态区间

| BTC.D 范围 | 状态 | 颜色 | 含义 |
|------------|------|------|------|
| > 60% | BTC 主导 | 🟠 橙色 | 保守配置，以 BTC 为主 |
| 45% - 60% | 均衡 | 🟡 黄色 | 可适度配置山寨 |
| < 45% | 山寨季 | 🟢 绿色 | 山寨活跃，注意风险 |

## 图表配置

### 系列
| 名称 | 颜色 | 默认显示 |
|------|------|----------|
| BTC.D | #f7931a (BTC橙) | ✅ |

### 阈值线
- 60% 线：橙色虚线
- 45% 线：绿色虚线

### 区域填充
- > 60%：橙色半透明
- < 45%：绿色半透明

### Y 轴范围
建议固定范围：30% - 100%（避免早期数据压缩显示）

### 默认显示范围
从 2014-01-01 开始

## 特殊注意

- 早期 BTC.D 接近 100%，2017 年后才有明显变化
- 稳定币市值增长会稀释 BTC.D，需考虑"剔除稳定币"的版本
- 数据可能因不同来源有差异
- 数值显示带 % 符号
