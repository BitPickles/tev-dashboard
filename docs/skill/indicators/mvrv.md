# MVRV 指标设计

## 指标概述

| 属性 | 值 |
|------|-----|
| 名称 | MVRV (Market Value to Realized Value) |
| 路径 | `/mvrv/index.html` |
| 数据 | `indicators/data/mvrv.json` |
| 用途 | 链上估值指标，判断 BTC 整体盈亏状态 |

## 计算公式

```
MVRV = 市值 (Market Cap) / 已实现市值 (Realized Cap)
```

### 市值 (Market Cap)
```
Market Cap = 当前价格 × 流通供应量
```

### 已实现市值 (Realized Cap)
```
Realized Cap = Σ(每个 UTXO 最后移动时的价格 × 数量)
```
即：按每个币最后一次移动时的价格计算的总市值。

## 指标含义

| MVRV | 含义 |
|------|------|
| > 1 | 平均持币处于盈利状态 |
| < 1 | 平均持币处于亏损状态 |
| 极高 (>3.5) | 市场过热，获利盘多 |
| 极低 (<1) | 市场恐慌，抄底机会 |

## 状态区间

| MVRV 范围 | 状态 | 颜色 | 含义 |
|-----------|------|------|------|
| < 1 | 低估 | 🟢 绿色 | 链上亏损，底部区域 |
| 1 - 2.4 | 正常 | 🟡 黄色 | 健康状态 |
| 2.4 - 3.5 | 偏高 | 🟠 橙色 | 开始过热 |
| > 3.5 | 过热 | 🔴 红色 | 顶部区域 |

## 图表配置

### 系列
| 名称 | 颜色 | 默认显示 | Y 轴 |
|------|------|----------|------|
| MVRV | #3b82f6 (蓝) | ✅ | 右轴 (线性) |
| BTC 价格 | #f7931a (橙) | ❌ | 左轴 (对数) |

### 阈值线
- 1.0 线：绿色虚线
- 2.4 线：橙色虚线
- 3.5 线：红色虚线

### 区域填充
- < 1：绿色半透明
- > 3.5：红色半透明

### 默认显示范围
从 2014-01-01 开始（数据实际从 2010-07-18 开始）

## 数据格式

```json
{
  "updated_at": "2026-02-12",
  "current": {
    "date": "2026-02-10",
    "value": 2.15,
    "btc_price": 68688
  },
  "history": [
    {
      "date": "2010-07-18",
      "value": 146.04,
      "btc_price": 0.09
    }
  ]
}
```

## 数据来源

- Glassnode (需要订阅)
- 或通过 BTC 价格 + UTXO 数据计算

## 特殊注意

- MVRV 是链上指标，数据更新可能有延迟
- 历史数据从 2010 年开始，是最早的指标之一
- BTC 价格使用对数轴显示
- 早期 MVRV 波动极大（>100），图表需要适当处理
