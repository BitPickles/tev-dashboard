# Crypto3D 数据站

基于「代币赋能价值 (TEV)」的加密货币数据平台。

## 线上地址

- **正式站**: https://crypto3d.pro
- **测试站**: https://bitpickles.github.io/tev-dashboard
- **GitHub**: https://github.com/BitPickles/tev-dashboard

## 核心指标

| 指标 | 路径 | 说明 |
|-----|------|------|
| TEV | /tev | 代币赋能价值，27个协议 |
| AHR999 | /ahr999 | 九神指标，BTC 定投时机 |
| MVRV | /mvrv | 链上估值，市值/已实现市值 |
| BMRI | /bmri | 宏观风险指数 |
| BTC.D | /btc-dominance | BTC 市值占比 |

## 数据更新

- **自动更新**: 每日 0:00 / 12:00 (AHR999, MVRV, BMRI, TEV 价格)
- **手动更新**: TEV 分配比例（月度核实）

## 技术栈

- 前端: 原生 HTML/CSS/JS + Chart.js
- 部署: Cloudflare Pages (正式站) + GitHub Pages (测试站)
- 数据: CoinGecko, DefiLlama, Coin Metrics, FRED
