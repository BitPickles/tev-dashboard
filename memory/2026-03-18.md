## 21:30 - Bug 修复 + Mantle 接入

### BNB BEP bug 修复
- **问题**：`unhashable type: 'slice'` + GitHub rate limit 403
- **原因**：rate limit 时返回 dict 而非 list，`resp.json()[:10]` 对 dict 做 slice 报错
- **修复**：
  1. 加 `GITHUB_TOKEN` 环境变量（从 `gh auth token` 获取）
  2. 检查 `isinstance(resp.json(), list)` 再做 slice
- **结果**：成功获取 10 个 BNB BEP 提案

### SSL 问题修复
- **问题**：Spark (`sparkfi.eth`) + Sky (`vote.makerdao.com`) 因 LibreSSL 2.8.3 报 SSL EOF 错误
- **修复**：对 SSL 失败的请求改用 `subprocess.run(curl)` 绕过 Python SSL
- **结果**：Spark 23 条、Sky 5 条提案正常获取
