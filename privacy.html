隐私声明

最后更新：2026-04-26

1. 本站是什么

Crypto3D（crypto3d.pro）为长期投资者提供价值判断工具。我们聚合来自 DefiLlama、CoinGecko、区块浏览器（Etherscan、BSCScan、TronScan 等）以及各公链 RPC 的公开链上数据，以表格、图表的形式展示给访问者。

本站的立场非常清晰：

  • 不是交易所——不撮合任何交易
  • 不是钱包——不请求连接任何钱包（MetaMask / WalletConnect / TronLink 等）
  • 不托管资产——不持有用户任何 token 或密钥
  • 不提供登录——没有账号系统

2. 我们收集什么

由于本站是公开数据看板，我们不收集任何可以标识你个人身份的信息（PII）。具体而言：

  • 不要求注册账号
  • 不要求你连接钱包
  • 不要求你输入邮箱、手机号、姓名、身份证号、KYC 资料
  • 不植入追踪像素（tracking pixel）
  • 不使用第三方广告网络

浏览器在访问任何网站时都会产生的基础请求信息（例如 IP 地址、User-Agent），由 Cloudflare 作为 CDN / 安全层在访问日志中短期保留，用于防 DDoS、识别爬虫和调试站点性能。这些日志遵守 Cloudflare 的隐私政策，我们不会尝试基于这些日志去识别具体个人。

3. 第三方资源

页面会向以下第三方发起请求：

  • Google Fonts——字体文件
  • DefiLlama / CoinGecko 等数据源——部分图表数据前端直接调用

这些第三方会看到你的 IP 和 User-Agent，他们各自的隐私政策适用。

4. Cookie / LocalStorage

本站不写入任何用于追踪的 Cookie。唯一使用的是浏览器本地存储（localStorage）中的几个键：

  • theme：记住你选择的亮色 / 暗色主题
  • lang：记住你选择的中 / 英语言

这些数据只在你浏览器本地，清除浏览器数据即可删除。

5. 数据准确性

站点展示的所有数据来源于公开链上记录和第三方聚合接口。我们努力保持准确，但不对数据的绝对准确性、实时性、完整性作任何担保。详见 风险披露。

6. 儿童

本站面向加密行业从业者和研究者，非面向 13 岁以下儿童。

7. 本声明的更改

若对本声明作实质性修改，我们会在本页顶部更新"最后更新"日期。
