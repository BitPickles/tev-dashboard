2. 加密资产的固有风险

加密资产具有高度投机性，历史数据反复表明：

  • 价格可能归零——无论是代币、NFT、LP 份额
  • 智能合约可能被攻击——导致资产永久损失
  • 协议可能 Rug Pull——团队跑路、协议废弃
  • 监管政策可能突变——导致服务停止或资产无法出金
  • 私钥遗失或被盗——资产不可恢复
  • 流动性可能消失——即便持仓有"市值"，也可能无法按该价成交

不要投入你不能承受完全损失的资金。

4. TEV / Token Economic Value 口径说明

TEV 是本站的核心原创指标，衡量 token 持有人从协议运转中捕获的经济价值。其计算：

  • 尽量优先采用链上直接可验证的数据（如 burn 地址持仓、买回钱包 USDT 流出）
  • 其次采用官方治理披露（如基金会 Treasury 报告）
  • 最后才采用第三方聚合口径（如 DefiLlama holdersRevenue），并在详情页说明口径差异

对于可能存在"做账式销毁"（pocket-to-pocket fake burn，例如源头和销毁目的地都属于同一方）的情形，我们会在详情页 Caveats 卡片中明确标注，并将相应的 TEV 归零。
